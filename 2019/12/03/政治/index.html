研途——政治

Posted on 2019-12-03 Edited on 2020-06-02 In 研途 Views:

反侵略战争（1840～1919）

鸦片战争（1840～1842）

  • 中英《南京条约》（第一个不平等条约）《五口通商章程》《虎门条约》
    中美《望厦条约》
    中法《黄埔条约》
  • 破坏了中国的主权和领土完整、领海主权、司法主权、关税主权
  • 逐步变成两半国家，逐渐开始反帝反封建的资产阶级民主革命

三元里人民抗英斗争（1841.5）

  • 广州，近代史上第一次大规模的反侵略武装斗争

爱国官兵的反侵略斗争

  • 第一次鸦片战争：关天培、陈化成、海龄
  • 第二次鸦片战争：史荣椿、乐善
  • 中日甲午战争：邓世昌、林永升

边境危机

  • 英国：从印度入侵西藏，又从缅甸入侵云南
  • 法国：从越南入侵广西
  • 俄国：从中亚入侵新疆
  • 日本：吞并琉球，入侵台湾

中日甲午战争（）

  • 中日《马关条约》
    割让台湾、澎湖列岛、辽东半岛给日本
俄国联合法国、德国，“干涉还辽”迫使日本放弃辽东半岛

义和团运动（中日甲午战争后）

  • 积极性
    在粉碎帝国主义列强瓜分中国的斗争中发挥了重大的历史作用
  • 局限性
    1.存在着笼统排外主义的错误
2.蒙受封建统治者的欺骗
3.存在迷信、落后的倾向

列强不能瓜分中国的原因

  • 根本：中华民族进行的不屈不挠的反侵略斗争
  • 重要：帝国主义列强之间的矛盾和相互制约

反侵略战争失败的原因

  • 根本原因：社会制度的腐败
  • 重要原因：经济技术落后（洋务运动-学习技术-中体西用）

重要会议

六届六中全会（1938年）

  • 毛泽东《论新阶段》，最早提出“马克思主义中国化”
  • 毛泽东《战争和战略问题》，确立了农村包围城市的革命道路理论

中共七大（1945年4月23日至6月11日）

  • 把“毛泽东思想”写进党章
  • 毛泽东《论联合政府》，进一步把新民主主义的政治、文化和经济与党的基本纲领联系起来
  • 毛泽东《论联合政府》，三大优良作风，理论联系实际、密切联系群众、批评与自我批评

七届二中全会（1949年3月5日至13日）

  • 毛泽东，两个务必：

十一届三中全会（1978.12）

  • 邓小平，重新确立了解放思想、实事求是的思想路线，停止使用“以阶级斗争为纲”的错误提法

党的十二大（1982）

  • “邓小平理论”的主题形成，“中国特色的社会主义”

党的十三大（1987）

  • 标志“着邓小平理论”轮廓的形成
    1.第一次比较系统地论述论文我国社会主义初级阶段理论
2.明确概括了党的“一个中心、两个基本点”的基本路线
3.第一次对中国特色社会主义理论的主要内容作了系统概括

南方谈话（1992）

  • “邓小平理论”走向成熟
  • 社会主义本质概括
    社会主义的本质，是解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕
  • 社会主义本质科学概括的意义
    1.既包含了社会主义社会的生产力问题，又包括了社会主义生产关系为基础的社会关系问题，是一个有机的整体
2.为我们坚持公有制又完善和发展公有制指出了明确的方向；两条根本原则（以社会主义公有制经济为主体，共同富裕）
3.遵循了科学社会主义的基本原则

党的十四大（1992）

  • 总结了改革开放以来十四年取得的巨大成就

党的十五大（1997）

  • 正式提出“邓小平理论”这一概念
  • 把“邓小平理论”写进党章（1999年邓小平理论载入宪法）

毛泽东的著作

《中国的红色政权为什么能够存在？》《井冈山的斗争》《星星之火，可以燎原》(1927)

  • 土地革命早期，初步形成了农村包围城市的革命道路理论

《论新阶段》（1938，六届六中全会）

  • 首次提出马克思主义中国化

《战争和战略问题》（1938,六届六中全会）

  • 确立了农村包围城市的革命道路理论

《⟨共产党人⟩发刊词》（1939.10.4）

  • 三大法宝
    统一战线，武装斗争，党的建设

《中国革命和中国共产党》（1939.12）

  • 第一次提出新民主主义革命的科学概念和总路线的内容
  • 无产阶级领导、人民大众的、反帝、反封建

《新民主主义论》（1940）

  • 阐述了新民主主义的政治、经济和文化

《论联合政府》（1945，党的七大）

  • 进一步把新民主主义的政治、文化和经济与党的基本纲领联系起来
  • 三大优良作风
    理论联系实际、密切联系群众、批评与自我批评

《在晋绥干部会议上的讲话》（1948）

  • 完整地表述了总路线的内容
  • 无产阶级领导、人民大众的、反帝、反封建、反官僚资本主义

《论人民民主专政》（1949）

  • 对新民主主义革命的基本经验作论集中概括

《论十大关系》（1956.4）

  • 马克思主义同中国实际第二次结合的标志
  • 确立基本方针
    努力把党内党外、国内国外的一切积极的因素，直接的、间接的积极因素全部调动起来，为社会主义建设服务

《如何处理人民内部的矛盾》（1957.2）补充修改后为《关于正确处理人民内部的矛盾》

  • 系统论述了社会主义社会矛盾的理论

    1.社会主义社会同样存在着矛盾
    2.社会主义基本矛盾的内容、性质和解决途径

    内容：生产关系和生产力之间的矛盾，上层建筑和经济基础之间的矛盾
性质：非对抗性的
解决途径：经过社会主义制度本身，不断得到解决

    3.关于社会主义社会的两类不同性质的矛盾

    政治上，分为敌我矛盾和人民内部的矛盾，两者在一定条件下可以相互转化
敌我矛盾：对抗性，采用专政
人民内部矛盾：非对抗性，采用民主
  • 用民主的方法解决人民内部的矛盾，是一个总方针，具体方针、原则：
    政治思想：团结-批评-团结
物质利益、分配：统筹兼顾、适当安排
人民群众和政府机关：坚持民主集中制
科学文化：百花齐放、百家争鸣（双百）
共产党和民主党派：长期共存、互相监督
民族之间：民族平等、团结互助
  • 中国工业化道路，重工业、轻工业和农业发展关系，走有别于苏联的中国工业化道路
    农业为基础，工业为主导，以农轻重为序发展国民经济的总方针
“两条腿走路”方针
