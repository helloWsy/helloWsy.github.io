首页显示文章浏览

  • 在文章内容处添加 <!-- more -->
  • 相比通过配置文件的好处是，首页的文章内容可以看到Markdown的渲染效果，而不再是显示源代码

数学公式启用

  • 启用 Mathjax，（之前版本的NexT还去网上搜各种插件，这里真简单…）

由于Markdown使用的 marked渲染引擎 和 Mathjax 在遇到 \ 和 _ 的时候会出现冲突

  • 可通过手动转义 \_ 和 \\，解决，但影响文章的移植性
  • 直接去掉 marked 中对 \ 和 _ 的转义，建议注释

添加评论功能

NexT自带很多评论的插件，在几番权衡之下决定使用 gitalk

  • 首先需要一个GitHub账号（如没有自行申请）

  • 新建一个OAuth应用

  • 启用 gitalk，并修改填写相应的内容

  • 在 http://localhost:4000 中无法正常使用，需要在发布的站点上使用

  • 如果有 about、tags、categroies等页面，需要在source\ _post\about\index.md 中添加 comments: false 关键词，否则评论框会出现在上述页面

文章版权声明

  • 配置 author 和 url

  • 启用版权声明 post: true

插入图片

在之前版本的NexT，好像是5.x，为了能够让图片缩小并居中，我直接使用<img>标签快速实现

在升级到 NexT 7.8 版本后，发现图片引用路径错误，浏览生成的 public 文件发现，生成的图片为同一目录，即src="photo.png"
之后发现 algin=center 无法实现图片居中，既然之前的偷懒方法用不了了，那就直接修改主题样式，使用 Markdown 语法来引用图片

  • 图片 title 显示并设置高度

  • 启用 fancybox
